Comment connecter Google Drive ?

AVP-Py attend le contenu complet d'un fichier rclone.conf. Ce contenu inclut le token d'autorisation Google : ne le partage jamais et ne l'envoie pas dans des captures d'écran ou des logs.

  1. Sur un ordinateur avec un navigateur, installe rclone.
  2. Lance rclone config dans un terminal.
  3. Choisis n pour créer un remote et nomme-le gdrive.
  4. Choisis le stockage drive (Google Drive).
  5. Sauf besoin particulier, laisse Client ID et Client Secret vides, choisis l'accès complet à Drive, puis refuse la configuration avancée.
  6. Accepte l'authentification automatique. Dans le navigateur, connecte le compte Google qui peut consulter le dossier de vidéos.
  7. Termine l'assistant, puis lance rclone config file pour afficher l'emplacement de rclone.conf.
  8. Ouvre ce fichier, copie tout le bloc [gdrive], puis colle-le dans le champ « Contenu complet de rclone.conf » ci-dessous.
  9. Garde gdrive comme nom du remote, indique le chemin du dossier, sauvegarde, puis clique sur « Tester la connexion ».

Exemple de structure seulement — la valeur réelle de token est générée par rclone :

[gdrive]
type = drive
scope = drive
token = {"access_token":"...","refresh_token":"..."}

Si le dossier a été partagé par quelqu'un d'autre, ajoute-le de préférence comme raccourci dans « Mon Drive » avant de renseigner son chemin.
